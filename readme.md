# Leo

一个简单实现react语法转换为微信小程序语法的项目。

> Leo，雷欧奥特曼，泰罗·奥特曼（Taro）的弟弟。

## 简介
本项目在阅读完[taro](https://github.com/NervJS/taro)相关源码后，仿照taro实现了一个简化版taro，主要实现react语法转换为微信小程序语法的功能，目的是让让大家可以用最简单代码了解taro的设计原理。 😊


## 如何使用
安装

```
npm install
```

编译
```
npm run build:weapp
```

编译完成直接用小程序开发工具打开dist目录即可
